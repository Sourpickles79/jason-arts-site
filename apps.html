3ds Max Modeling Tool · Version 1.0

PolyRegularizer

Flatten selected polygons, regularize their boundary, or combine both operations in one controlled pass. Adjustable strength and automatic boundary discovery make cleanup faster without turning a focused task into a complicated workflow.

Inspired by Garp's original Loop Regularizer v1.1 on ScriptSpot, a respected modeling script that had not been updated since 2011. PolyRegularizer carries that useful idea forward with compatibility across 3ds Max versions and an expanded polygon-based workflow.

Three focused operations

Flatten and regularize together, regularize only the polygon boundary, or flatten the selected polygons without changing the boundary.

Controlled results

Dial the operation from 0–100 strength, use a single Editable Poly selection, and keep the entire change inside one undo step.

Flexible installation

Use the drag-and-drop MZP installer, install the toolbar macro manually, or inspect and run the included readable MaxScript source.

Quick installation

  1. Download and extract the release ZIP.
  2. Drag the included .mzp file into a 3ds Max viewport, or copy the .mcr file into your userMacros folder.
  3. Open Customize User Interface → Toolbars, select JasonArts Tools, and add Poly Regularizer to a toolbar.
  4. Use a collapsed Editable Poly, enter Polygon sub-object mode, select polygons, and run the tool.
